函数包名称

pymysql

数据连接对象

参数名 类型 说明
host 字符串 mysql服务器地址
port 数字 mysql服务器端口号
user 字符串 用户名
password 字符串 密码
db 字符串 数据库名称
charset 字符串 连接编码
connection对象支持的方法
方法名 说明
cursor() 使用该连接创建并返回游标
commit() 提交当前事务
roolback() 回滚当前事务
close() 关闭连接
游标对象
参数名 说明
execute() 执行命令
fetchone() 取结果集的下一行
fetchmany() 获取结果集的下几行
fetchall() 获取结果集中剩下的所有行
rowcount() 最近一次execute返回数据的行数或影响行数
close() 关闭游标对象

数据库查询操作

  • fetchone()：结果集是一个对象
  • fetchall()：结果集是所有对象
  • fetchmany()结果集是指定个数。例如：fetchmany(10)就是10个对象
import pymysql

db=pymysql.connect(host="localhost",user="root",
                   port=3306,passwd="282515",
                   db="sql_learing",charset="utf8")

cursor=db.cursor()
sql_01="select 名字,评分 from hotel where 评分 > %s"%(2)

try:
    cursor.execute(sql_01)
    # 拿到所有行的数据存到result中
    result=cursor.fetchall()
    for row in result:
        hotel_name=row[0]
        hotel_score=row[1]
        print("%s 的评分是 %s"%(hotel_name,hotel_score))
except:
    print("error")

事务处理

对于支持事务处理的数据库，在Python数据库编程中，当游标建立之时，就自动开始了一个隐形的数据库事务。

import pymysql

db=pymysql.connect(host="localhost",user="root",
                   port=3306,passwd="282515",
                   db="sql_learing",charset="utf8")

cursor=db.cursor()
sql_01="update t2 set x=%s where x=%s"%(111,3)
sql_02="insert into t2 values(4,4000)"
sql_03="delete from t2 where x=%s"%(1)

try:
    cursor.execute(sql_01)
    print(cursor.rowcount)
    cursor.execute(sql_02)
    print(cursor.rowcount)
    cursor.execute(sql_03)
    print(cursor.rowcount)
    # 如果都可执行，则提交
    db.commit()
except:
    # 否则rollback
    db.rollback()
    print("error")

实例：银行转账系统

import pymysql

class transfermoney():
    def __init__(self,conn):
        self.conn=conn
        
    # 检查账户是否可用
    def check_account_available(self,acctid):
        cursor=self.conn.cursor()
        try:
            sql="select * from t2 where x=%s"%acctid
            cursor.execute(sql)
            print("check_account_available:"+sql)
            result=cursor.fetchall()
            if len(result)!=1:
                print("账号%s不存在"%acctid)
        finally:
            cursor.close()
            
    # 检查账户里是否有足够的钱
    def has_enough_money(self,acctid,money):
        cursor=self.conn.cursor()
        try:
            sql="select * from t2 where x=%s and y>%s"%(acctid,money)
            cursor.execute(sql)
            print("has_enough_money:"+sql)
            result=cursor.fetchall()
            if len(result)!=1:
                print("账号%s没有足够的钱"%acctid)
        finally:
            cursor.close()

    # 扣钱
    def reduce_money(self,acctid,money):
        cursor=self.conn.cursor()
        try:
            sql="update t2 set y=y-%s where x=%s"%(money,acctid)
            cursor.execute(sql)
            print("reduce_money:"+sql)
            if cursor.rowcount!=1:
                print("账号%s扣款失败"%acctid)
        finally:
            cursor.close()

    # 加钱
    def add_money(self,acctid,money):
        cursor=self.conn.cursor()
        try:
            sql="update t2 set y=y+%s where x=%s"%(money,acctid)
            cursor.execute(sql)
            print("reduce_money:"+sql)
            if cursor.rowcount!=1:
                print("账号%s加款失败"%acctid)
        finally:
            cursor.close()

    # 开始转账（处理事务）
    def transfer(self,source_id,target_id,money):
        try:
            self.check_account_available(source_id)
            self.check_account_available(target_id)
            self.has_enough_money(source_id,money)
            self.reduce_money(source_id,money)
            self.add_money(target_id,money)
            self.conn.commit()
        except:
            self.conn.rollback()
            print("转账失败")

# 创建连接
conn=pymysql.connect(host="localhost",port=3306,user="root",passwd="282515",
                     db="sql_learing",charset="utf8")
# 输入账户和钱

source_id=input()
target_id=input()
money=input()

tr_money=transfermoney(conn)
try:
    tr_money.transfer(source_id,target_id,money)
except:
    print("error")
finally:
    conn.close()
