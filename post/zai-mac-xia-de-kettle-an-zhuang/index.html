安装Java的JDK

https://www.oracle.com/java/technologies/javase-jdk8-downloads.html

选择对应地址，比如我的事MacOS就选择下图选项。下载好后，直接安装，直到最后一步就可以了。

验证下载是否成功

在终端输入：

java version

如果显示为下图，则表示安装成功

启动Kettle

在终端输入Kettle文件夹的地址并回车。比如我的地址是：

 /Users/zhaoguangyao/data-integration

然后输入以下命令并回车：

sh spoon.sh

弹出Spoon-Kettle程序框

安装Mysql驱动

连接Mysql数据库时会报错提示安装驱动。驱动的安装地址如下：

https://link.jianshu.com/?t=https://dev.mysql.com/downloads/file/?id=468318%20

下载后安装解压，然后将如图所示jar文件（两个中的任意一个）拷贝到data-integeation下的lib文件夹中。

注意：必须要用5.X版本的，不能用8.X！本人就遇到了这个坑。
