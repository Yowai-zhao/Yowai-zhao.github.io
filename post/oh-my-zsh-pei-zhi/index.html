oh-my-zsh配置

2020-03-15 2 min read

下载iterm2

官网：https://www.iterm2.com/

或者直接在App store安装

切换shell为zsh

chsh -s /bin/zsh

验证是否切换成功

echo $SHELL

安装oh-my-zsh

sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

修改主题配置

查看主题种类

主题都在此文件夹存放

浏览主题样式

https://github.com/ohmyzsh/ohmyzsh/wiki/Themes

修改主题

打开家目录下的「.zshrc」文件，然后修改配置。「.zshrc」文件是隐藏文件，如果用访达打开的话，需要用「shift+command+.」显示隐藏文件。其中灰色即为隐藏文件。

打开「.zshrc」文件，修改如图所示信息保存，然后重启客户端即可。

命令高亮

zsh-syntax-highlighting地址：https://github.com/zsh-users/zsh-syntax-highlighting

 git clone https://github.com/zsh-users/zsh-syntax-highlighting.git
 echo "source ${(q-)PWD}/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh" >> ${ZDOTDIR:-$HOME}/.zshrc

重启客户端，即可启动高亮效果
